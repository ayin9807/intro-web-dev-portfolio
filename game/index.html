Welcome to Breakout!

The objective of this game is to hit as many bricks as possible. Once all 20 have been hit, the game will replenish with new bricks. You only have 1 life, so be careful!

Use the left and right keys to move the paddle.

Also, if you lose, the game will automatically start back up again, so be prepared!!
